## Screenshot Specifications

### Recommended Settings
- **Resolution**: 1920x1080 (desktop), 1440x900 (alternative)
- **Format**: PNG (for quality), JPG (for file size)
- **Browser**: Chrome or Firefox (latest version)
- **Device**: Use browser DevTools for responsive screenshots

### Tools for Screenshots
- **Browser DevTools**: Built-in responsive design mode
- **Screenshot Tools**: 
  - Lightshot
  - Snagit
  - Browser extensions
- **Screen Recording**: 
  - OBS Studio (free)
  - Loom (easy sharing)
  - QuickTime (Mac)
  - Windows Game Bar (Windows)

### GIF Creation Tools
- **ScreenToGif** (Windows, free)
- **Giphy Capture** (Mac, free)
- **Kap** (Mac, free)
- **Online Tools**: Ezgif.com, CloudConvert

## Video Walkthrough Script

### Suggested Flow (2-3 minutes)

1. **Introduction (10s)**
   - Show homepage
   - "This is Neighborly, a unified marketplace for local services"

2. **Search Demo (30s)**
   - Type in search bar: "pet sitter in 90210"
   - Show results loading
   - Highlight filter sidebar

3. **Filtering (30s)**
   - Apply price filter
   - Select rating filter
   - Show results updating

4. **Provider Details (40s)**
   - Click on provider card
   - Show modal opening
   - Navigate through tabs (Overview â†’ Experience â†’ Portfolio â†’ Reviews)

5. **Platform Filter (20s)**
   - Click "Care.com" tab

6. **Mobile View (20s)**
