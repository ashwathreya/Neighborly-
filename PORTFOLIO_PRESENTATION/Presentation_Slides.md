## SLIDE 12: Challenges & Solutions

### Content:
**Title:** Key Challenges & Solutions

**Challenge 1: Multi-Platform Aggregation**
- **Problem:** 35+ platforms with different data structures
- **Solution:** Unified data model with platform-specific adapters
- **Result:** Consistent UI, easy to add new platforms

**Challenge 2: Real-Time Filtering Performance**
- **Problem:** Multiple filters causing lag
- **Solution:** Debouncing, memoization, optimized logic
- **Result:** Smooth filtering, < 100ms response time

**Challenge 3: Location-Based Distance**
- **Problem:** Calculate distances from ZIP codes
- **Solution:** Geocoding API + Haversine formula
- **Result:** Accurate distances, better user experience

**Challenge 4: Dynamic Backgrounds**
- **Problem:** Service-specific visuals without hardcoding
- **Solution:** Unsplash API with dynamic queries
- **Result:** Contextual, engaging visuals

### Graphics:
- **Visual:** Challenge → Solution → Result flow for each
- **Icons:** Problem icon, solution icon, checkmark
- **Screenshots:** Before/after comparisons
