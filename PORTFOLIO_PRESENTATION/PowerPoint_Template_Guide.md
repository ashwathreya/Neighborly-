# PowerPoint Template Setup Guide

## How to Create Your Presentation

This guide provides step-by-step instructions for creating your Neighborly presentation in PowerPoint using the slide content provided.

---

## STEP 1: Create New Presentation

1. Open PowerPoint
2. Choose a template:
   - **Recommended:** Blank presentation or "Modern" template
   - Avoid: Busy templates that distract from content
3. Set slide size: **16:9 (Widescreen)**

---

## STEP 2: Master Slide Setup

### Design Theme
- **Background:** White or very light gray
- **Accent Colors:** 
  - Primary: Purple (#6366f1)
  - Secondary: Pink (#ec4899)
  - Gradient: Purple → Pink
- **Font:** Inter (or similar: Calibri, Arial)

### Slide Master Customization
1. Go to View → Slide Master
2. Set default fonts:
   - Headings: Bold, 36-48pt
   - Body: Regular, 18-24pt
3. Add footer (optional): Slide number, your name
4. Set default colors in theme colors

---

## STEP 3: Slide-by-Slide Creation

### For Each Slide:

1. **Add Title Slide (Slide 1)**
   - Insert → New Slide → Title Slide
   - Add title: "Neighborly"
   - Add subtitle: "Multi-Platform Service Marketplace"
   - Add your name and date
   - Insert background image (collage of services)
   - Add gradient overlay (purple to pink, 50% opacity)

2. **Content Slides (Slides 2-19)**
   - Insert → New Slide → Title and Content
   - Copy content from `Presentation_Slides.md`
   - Format headings (36pt, bold)
   - Format body text (18-24pt)
   - Add bullet points where needed

3. **Thank You Slide (Slide 20)**
   - Insert → New Slide → Blank
   - Center "Thank You" text (72pt)
   - Add contact information below
   - Add social media icons

---

## STEP 4: Add Graphics

### Inserting Screenshots:
1. Insert → Pictures → This Device
2. Select your screenshot
3. Resize to fit slide
4. Add border (optional): Picture Format → Picture Border
5. Add shadow for depth: Picture Format → Picture Effects → Shadow

### Inserting Icons:
1. Insert → Icons (PowerPoint 365) or Insert → Shapes
2. Search for icon or use shapes
3. Color: Use theme colors
4. Size: Consistent (64x64px or 128x128px)

### Creating Diagrams:
1. Insert → SmartArt (for flowcharts)
2. Or Insert → Shapes (for custom diagrams)
3. Use consistent colors
4. Add arrows: Insert → Shapes → Arrow
5. Group elements: Select all → Right-click → Group

### Adding Charts:
1. Insert → Chart
2. Choose chart type (Bar, Line, Pie)
3. Enter data from `Presentation_Slides.md`
4. Format: Use theme colors
5. Add data labels

---

## STEP 5: Design Consistency

### Color Scheme Application:
- **Headings:** Purple gradient or dark gray
- **Accent Elements:** Purple/Pink gradient
- **Backgrounds:** White or light gray
- **Text:** Dark gray (#111827) or black
- **Highlights:** Platform colors for specific elements

### Typography Consistency:
- **Slide Titles:** 36-48pt, Bold
- **Section Headings:** 24-32pt, Semi-bold
- **Body Text:** 18-24pt, Regular
- **Captions:** 14-16pt, Regular
- **Labels:** 12-14pt, Semi-bold

### Spacing:
- **Margins:** Consistent (0.5-1 inch)
- **Element Spacing:** Use grid (align to grid)
- **Line Spacing:** 1.2-1.5 for body text

---

## STEP 6: Animations & Transitions

### Slide Transitions:
1. Select slide
2. Transitions → Choose "Fade" or "Morph"
3. Duration: 0.5-1 second
4. Apply to all slides (optional)

### Object Animations (Use Sparingly):
1. Select object
2. Animations → Choose effect
3. Recommended:
   - Fade In (for text)
   - Fly In (for diagrams)
   - Zoom (for images)
4. Duration: 0.3-0.5 seconds
5. Start: On Click or With Previous

### Animation Tips:
- Keep it subtle
- Don't overuse
- Test timing
- Ensure smooth playback

---

## STEP 7: Slide Layouts

### Recommended Layouts:

**Title Slide:**
- Full-bleed background image
- Centered text overlay
- Gradient overlay for readability

**Content Slides:**
- Title at top (left-aligned)
- Content in two columns (if needed)
- Images on right, text on left (or vice versa)
- Footer space for notes

**Comparison Slides:**
- Split screen layout
- Before on left, After on right
- Clear labels

**Statistics Slides:**
- Large numbers prominently displayed
- Icons or charts supporting data
- Clean, uncluttered layout

---

## STEP 8: Adding Notes

### Speaker Notes:
1. View → Notes Page
2. Add talking points for each slide
3. Include:
   - Key points to mention
   - Transition phrases
   - Time estimates
   - Questions to ask

### Example Notes Format:
```
Key Points:
- Mention 35+ platforms
- Highlight time savings
- Show screenshot

Transition:
"Now let's look at the technical implementation..."

Time: 2 minutes
```

## STEP 10: Export & Sharing

### Export Options:

1. **PDF (For Sharing):**
   - File → Export → Create PDF
   - Quality: High
   - Include: All slides, notes (optional)

2. **Video (For Online):**
   - File → Export → Create a Video
   - Quality: HD (1080p)
   - Include: Timings and narrations (if recorded)

3. **Images (For Portfolio):**
   - File → Export → Change File Type → PNG
   - Export all slides as images
   - Use for online portfolio

### File Naming:
- `Neighborly_Portfolio_Presentation.pptx`
- `Neighborly_Portfolio_Presentation.pdf`
- `Neighborly_Slide_01.png` (for individual slides)

## TIPS FOR BEST RESULTS

### Design Tips:
1. **Less is More:** Don't overcrowd slides
2. **Visual Hierarchy:** Use size, color, position
3. **Consistency:** Same fonts, colors, spacing
4. **Whitespace:** Give content room to breathe
5. **Contrast:** Ensure text is readable

### Content Tips:
1. **One Idea Per Slide:** Keep it focused
2. **Bullet Points:** Use sparingly (3-5 max)
3. **Visuals First:** Images > Text when possible
4. **Tell a Story:** Connect slides with narrative
5. **Practice:** Know your content

### Technical Tips:
1. **File Size:** Optimize images (compress if needed)
2. **Fonts:** Embed fonts or use system fonts
3. **Links:** Test all hyperlinks
4. **Backup:** Save multiple versions
5. **Compatibility:** Test on different versions of PowerPoint

## TEMPLATE CUSTOMIZATION

### If Using a Template:
1. Choose minimal template
2. Remove template graphics that don't fit
3. Customize color scheme to match Neighborly
4. Replace template fonts with your choices
5. Adjust layout to fit your content

### Custom Template Creation:
1. Design → Slide Size → Custom
2. Set dimensions: 1920x1080 (16:9)
3. Design → Format Background
4. Set background color/image
5. View → Slide Master
6. Customize master slides
7. Save as template (.potx)

---

## PRESENTATION MODE SETUP

### Before Presenting:
1. **Rehearse:**
   - Slide Show → Rehearse Timings
   - Practice transitions
   - Time your presentation

2. **Presenter View:**
   - Slide Show → Set Up Slide Show
   - Enable "Use Presenter View"
   - Shows notes, next slide, timer

3. **Backup:**
   - Save PDF version
   - Have screenshots ready
   - Test on presentation computer
